AMIWebFramework Docker Image
============================

The AMI Web Framework (AWF) is a framework for developing metadata-oriented applications. It was originally developed for the A Toroidal LHC ApparatuS (ATLAS) experiment, one of the two general-purpose detectors at the Large Hadron Collider (LHC).

This is the official Docker image.
